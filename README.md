# Registro para conferencia de tecnología

Este proyecto se basa en el famoso curso de [Manning.com](https://www.manning.com/livevideo/modern-php-web-development-with-mysql-github-and-heroku) llamado " Modern PHP Web Development with MySQL, GitHub and Heroku". En mi caso haré algunas modificaciones al proyecto enfocados en el diseño y lugar donde se desplegará, en este caso en AWS usando el servicio "AWS Lightsail".


## Técnologías usadas

- PHP
- MySQL
- HTML
- CSS
- Boostrap
- AWS Lightsail
